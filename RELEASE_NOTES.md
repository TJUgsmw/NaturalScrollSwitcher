# NaturalScrollSwitcher 0.5.0

Improves scroll reliability by correcting ordinary mouse wheel events directly instead of relying only on delayed global macOS setting changes.

## What's included

- Event-level correction for ordinary mouse wheel scrolling.
- Trackpad-first system baseline: macOS natural scrolling is kept aligned with the trackpad preference.
- Improved classification for ordinary USB/Bluetooth wheel mice.
- Menu diagnostics for recent input source and recent correction action.
- Existing custom app icon, menu bar icon, bilingual menu text, and drag-to-Applications DMG.

## Install

Download the `.dmg` or `.zip`, open `NaturalScrollSwitcher.app`, and grant both Input Monitoring and Accessibility permissions when macOS asks.

This app is ad-hoc signed for local use, not notarized by Apple. macOS may show a first-run warning.
